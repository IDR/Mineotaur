Welcome to Mineotaur's documentation!
=====================================

Mineotaur is a web application to share and visually analyse high-throughput/high-content microscopy screens developed at the Carazo Salas lab of the University of Cambridge, Department of Genetics.

The project website can be found at http://www.mineotaur.org. Please cite the following paper when using Mineotaur:
B. Antal, A. Chessel, R. E. Carazo Salas: Mineotaur: interactive visual analytics for high-content microscopy screens, under revision.
